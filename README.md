## Installation

1. Clone this repository
    ```
    git clone git@github.com:melonproject/website.git
    cd welcome

    ```

2. Install dependencies:
    ```
    meteor npm install
    ```

## Getting started

After installation is complete, go to the above `website` directory, open a terminal and launch meteor:

  ```
  meteor
  ```
